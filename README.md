**NeoPixel Strip Animations**

I created this project in 2024 to make my living room a bit more lively for Diwali.  
It uses an Arduino Uno and a NeoPixel strip to display colourful animations and effects.  

**Features**

- Colour wipes (red, green, blue, orange, purple)  
- Theatre-style chasing lights  
- Rainbow effects and rainbow cycling  
- Adjustable brightness and speed  

**Hardware**

- Arduino Uno  
- WS2812 NeoPixel LED strip  
- Appropriate power supply for the LED strip (I just used the Uno's 5V pin)

**Usage**

1. Open `NEOPIXEL_STRIP_FINAL.ino` in the Arduino IDE.  
2. Connect your Arduino and LED strip on Digital Pin 7.  
3. Upload the sketch to the Arduino.  
4. Watch your NeoPixel strip come alive with colourful effects!
